# Changelog

All notable changes to this project will be documented in this file.

## [v0.0.0] - 2026-02-02
### Added
- Core SVG editing engine (Grouping, Ungrouping, Color modification, Deletion).
- Multi-format support (SVG, EPS) with PDF/SVG/EPS export capabilities.
- Advanced Image Tracing with multiple presets (Default, Detailed, Simplified, Sketch).
- Robust Undo/Redo system with 50-step history.
- Modern UI with multi-selection canvas and context menus.
- Bilingual localization (English & Korean) with automatic system detection.
- Full CI/CD pipeline integrated with GitHub Actions.
- Automated multi-OS binary releases (Linux, Windows, macOS).
- Docker support via GitHub Container Registry (GHCR).
- Automatic versioning system based on commit counts.

# 변경 이력 (Changelog)

## [v0.0.0] - 2026-02-02
### 추가됨
- 핵심 SVG 편집 엔진 (그룹화, 그룹 해제, 색상 수정, 삭제).
- 다중 포맷 지원 (SVG, EPS) 및 PDF/SVG/EPS 내보내기 기능.
- 다양한 프리셋을 지원하는 고급 이미지 추적 기능 (이미지 트레이스).
- 50단계 히스토리를 지원하는 강력한 실행 취소/다시 실행 시스템.
- 다중 선택 및 컨텍스트 메뉴를 지원하는 현대적인 GUI.
- 영어 및 한국어 다국어 지원 (시스템 언어 자동 감지).
- GitHub Actions 통합 CI/CD 파이프라인 구축.
- 3대 OS(Linux, Windows, macOS)용 바이너리 자동 빌드 및 릴리스 기능.
- GitHub 컨테이너 레지스트리(GHCR)를 통한 도커 배포 지원.
- 커밋 횟수 기반 자동 버전 관리 시스템 도입.
