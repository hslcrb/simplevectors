# SimpleVectors

A modular vector graphics editor capable of opening, saving, converting, and modifying vector files (SVG, EPS).

## Features
- **Open & Save**: Support for SVG and EPS formats.
- **Conversion**: Seamless conversion between supported vector formats.
- **Editing**: Change colors, group elements, and export specific layers.
- **Internationalization**: Full support for English and Korean, detecting system default with manual toggle.
- **Modular Design**: Built with maintainability in mind.

# SimpleVectors (심플벡터)

SVG, EPS 등 벡터 파일을 열고, 저장하고, 변환하며 수정할 수 있는 모듈형 벡터 그래픽 편집기입니다.

## 기능 (Features)
- **열기 및 저장**: SVG 및 EPS 형식을 지원합니다.
- **변환**: 지원되는 벡터 형식 간의 매끄러운 변환이 가능합니다.
- **편집**: 색상 변경, 요소 그룹화, 특정 레이어 개별 내보내기가 가능합니다.
- **국제화**: 영어 및 한국어를 완벽하게 지원하며, 시스템 기본 설정을 감지하고 수동으로 전환할 수 있습니다.
- **모듈형 디자인**: 유지보수가 용이하도록 설계되었습니다.
